# Sample Data

This folder contains a redacted cached sample result used only when `GEMINI_API_KEY` is not configured or when a user explicitly clicks **Load Redacted Sample Result**.

The sample is not a live Gemini response. It recreates the reimbursement-review shape from earlier redacted demo data so judges can inspect UI behavior without uploading documents or using a secret key.

Do not place private financial documents, real API keys, unredacted invoices, or Streamlit secrets in this folder.
